# INSTALLATION

requires Python 3

git clone https://github.com/gotr00t0day/IGF.git

cd IGF

chmod +x igf.py<br/>
chmod +x main.py

pip3 install -r requirements.txt


# USAGE

./main.py

Config: To change domain / ip just type config and it will prompt you to enter the domain / ip you want to use.
